# Widget-Clock

Clock digital for Windows 10/11

![alt clock](1.2.png)
---
Is configurable

![alt menu](2.2.png)
---
Configs:

|Combination|Result|
|-----------|------|
|[True, True]| 24 hours with seconds|
|[True, False]| 24 hours no seconds|
|[False, True]| 12 hours with seconds|
|[False, False]| 12 hours no seconds|

![alt configs](3.png)

## Install
execute widget-clock-setup.exe

> with Python 3.12
